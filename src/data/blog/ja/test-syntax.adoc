== テキストフォーマット

*太字 (Bold)* と _イタリック (Italic)_ と `等幅フォント (Monospace)`。
これは #ハイライト (Highlight)# されたテキストです。
上付き文字: E = mc^2^
下付き文字: H~2~O

== リンク

http://asciidoctor.org これは自動的にリンクになりますし，
http://asciidoctor.org[AsciiDoctor] とすれば表示テキストを指定できます．

== リスト

=== 番号なしリスト

* アイテム 1
* アイテム 2
** インデントされたアイテム
* アイテム 3

=== 番号付きリスト

. ステップ 1
. ステップ 2
.. サブステップ a
.. サブステップ b
. ステップ 3

=== チェックリスト

* [x] 完了タスク
* [ ] 未完了タスク

=== 定義リスト

用語:: 定義の説明
AsciiDoc:: 軽量マークアップ言語のひとつ
複数行の定義::
これは定義の1行目です。
これは定義の2行目です。

== 引用 (Blockquote)

[quote, アインシュタイン, アルバート]
____
何かを学ぶためには、自分で体験する以上にいい方法はない。
____

[quote, 作者不明]
____
引用文のみのパターン。
____

== Verse Block (詩など)

[verse, 松尾芭蕉, 奥の細道]
____
古池や
蛙飛びこむ
水の音
____

== Example Block

.Exampleのタイトル
====
これはExample Blockです。

== Keyboard Macro

キーボードショートカットを表示します。
kbd:[Ctrl+C] でコピー、kbd:[Ctrl+V] で貼り付け。
kbd:[Ctrl+Shift+N] で新しいウィンドウを開く。

== アドモニション (注釈)

NOTE: これはノートです。

TIP: これはヒントです。

IMPORTANT: これは重要事項です。

WARNING: これは警告です。

CAUTION: これは注意書きです。

.ブロック形式のアドモニション
[NOTE]
====
これはブロック形式のノートです。
複数行に渡るコンテンツを含むことができます。

* リストアイテム
* リストアイテム
====

== 折りたたみブロック (Collapsible)

.クリックして詳細を表示
[%collapsible]
====
ここに隠されたコンテンツが表示されます。
詳細なデータや、長すぎるコードなどを隠すのに便利です。

[source,javascript]
----
console.log("Hidden code");
----
====

== コードブロック

.TypeScriptの例
[source,typescript]
----
interface User {
  name: string;
  id: number;
}

const user: User = {
  name: "Hayes",
  id: 0,
};
----
